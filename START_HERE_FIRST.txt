--------------------------

STEP 2: Update .env File
-------------------------
1. Go to: C:\Users\Ashish\Desktop\realty-collective
2. Open file: .env (with Notepad)
3. Find this line: DATABASE_URL=

STEP 3: Run Setup
-----------------
1. Right-click in the realty-collective folder
2. Click "Open in Terminal" or "Open PowerShell here"
