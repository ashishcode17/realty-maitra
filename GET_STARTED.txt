  3. Update .env with: postgresql://postgres:YOUR_PASSWORD@localhost:5432/realty_collective


STEP 2: CONFIGURE .env FILE
-----------------------------
Open .env file and make sure it has:

DATABASE_URL="your-database-connection-string"
JWT_SECRET="any-secret-key-12345"
NEXT_PUBLIC_APP_URL="http://localhost:3000"


STEP 3: RUN SETUP COMMANDS
---------------------------
Open PowerShell in realty-collective folder and run:

  npm run db:generate
  npm run db:migrate
  npm run db:seed
  npm run dev


STEP 4: OPEN BROWSER
--------------------
